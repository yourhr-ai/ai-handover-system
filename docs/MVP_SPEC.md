# AI 인수인계 자동화 시스템 MVP 개발 명세서 v1.0

## 개발 방식
- Windows EXE 프로그램
- 로컬 PC에서 실행
- OpenAI API 사용
- 로그인 기능 없음
- 사용자 관리 없음
- 데이터베이스 없음(MVP)
- 결과물은 Word(.docx) 저장

## 기술 스택

| 영역 | 기술 |
|------|------|
| UI | Python + PySide6 |
| AI | OpenAI API |
| 문서 생성 | python-docx |

## 저장 기능
- **필수**: 업무 인수인계 보고서.docx 생성
- **나중 버전**: docx / pdf / md 선택 저장

---

## 화면 명세

### 메인 화면

```
[업무분장 입력]
멀티라인 텍스트박스

[평가자료 업로드]
파일 선택 버튼
지원 형식: pdf / docx / xlsx / txt

[업무 복원 시작]
버튼

[분석 결과]
마크다운 형식 미리보기

[Word 저장] [Word 열기]
```

---

## 처리 흐름

1. 사용자 업무분장 입력
2. 평가자료 업로드
3. 평가자료 텍스트 추출
4. 업무분장 + 평가자료를 AI 프롬프트에 전달
5. GPT 응답 수신
6. 응답을 마크다운으로 화면 표시
7. Word(.docx) 생성
8. 사용자 저장

---

## MVP 범위

### 포함
- 업무분장 입력
- 평가자료 업로드
- OpenAI API 호출
- 결과 미리보기
- Word 저장

### 제외
- 이메일 분석
- 카카오톡 분석
- 폴더 분석
- 사용자 관리 / 로그인
- 클라우드 저장 / DB

---

## OpenAI 설정

- 기본 모델: `gpt-5.5`
- API Key: `.env` 파일의 `OPENAI_API_KEY` 환경변수 사용
- API Key가 없으면 실행 시 입력 요청 다이얼로그 표시
- 평가자료가 길 경우 자동 요약 후 분석
