# Handover Analyzer MVP 1

Windows desktop application skeleton for analyzing handover documents.

## Tech Stack

- Python
- PySide6
- python-docx

## Run

```powershell
pip install -r requirements.txt
python app/main.py
```

## MVP 1 Scope

- Minimal PySide6 main window
- Placeholder button actions only
- No document analysis logic
- No JSON or Word export logic
